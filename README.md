# Enumerable

> The Enumerable mixin provides collection classes with method of `all?`, `any?`, `filter`, `max`, `min`.

This project was built using:
- Ruby

## Getting Started
### Setup
- ```git clone https://github.com/ridaarif98/Enumerable.git```
- ```cd Enumerable```

### Usage
- ``` ruby test.rb```
